NXT I2C Library

NXTServo.setPosition()

Description

Run a Servo to a certain position

Syntax

setPosition(uint8_t number, uint8_t position);

Parameters

uint8_t number: The number of the servo you want to control.  Servo_1, 2, ..., 8 

uint8_t position: The position you want to set the servo to.  [500, 2500] or Servo_Position_Default

Returns

None
