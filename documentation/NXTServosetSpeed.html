NXT I2C Library

NXTServo.setSpeed()

Description

Set the speed of a specific Servo

Syntax

setSpeed(uint8_t number,  uint8_t speed);

Parameters

uint8_t number: The number of the servo you want to control.  Servo_1, 2, ..., 8 

uint8_t speed: The speed you want the servo to run at.  [0, inf] or Servo_Speed_Full

Returns

None
