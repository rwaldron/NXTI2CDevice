NXT I2C Library

This library allows an Arduino board to connect to NXT sensors and controllers.

Controllers

NXTMMX class

Able to connect and control two NXT motors with Arduino.

  • getBatteryVoltage()
  • getEncoderPosition()
  • reset()
  • runUnlimited()
  • runSeconds()
  • runTachometer()
  • runDegrees()
  • runRotations()

NXTServo Class

NXTServo-v2 allows you to control speed and position of up to 8 RC servo motors (or mini/micro RC servos)

  • runServo()
  • setSpeed()
  • setPosition()
  • BatteryVoltage()
  • reset()
  • storeInitial()
  • halfMacro()
  • resumeMacro()
  • gotoEEPROM()
  • editMacro()

PFMate class

PF Motor controller for NXT is designed to be a closely integrated controller for PF motors

  • sendSignal()
  • controlMotor()
  • issueCommand()

NXTHID class

Send HID input events (such as keyboard key presses) to host computer. These events can be received by any program that can run on your PC.

  • transmitData()
  • asciiMode()
  • directMode()
  • setModifier()
  • sendKeyboardData()


Sensors

ACCLNx class

This tiny and lightweight 3 axis accelerometer has multiple sensitivity levels of gravity. Use as a Tilt sensor, or measuring velocity or accelerations.

  • getXTilt()
  • getYTilt()
  • getZTilt()
  • getXAccl()
  • getYAccl()
  • getZAccl()
  • getSens()

DISTNx class

High Precision Infrared distance sensor.

  • getDist()
  • getVolt()
  • getType()

PSPNx class

Control your Arduino robot using a Sony PlayStation 2 Controller joystick. (or any other PS2 compatible remote control/joystick).

  • getXLJoy()
  • getYLJoy()
  • getXRJoy()
  • getYRJoy()
  • getButtons()

LineLeader class

Has an array of 8 sensors with a controlled light source, to detect a line.

  • getSteering()
  • getAverage()
  • getResult()

NXTPowerMeter class

Measure the power consumption of your robot (or any other device) while in operation.

  • resetCounters()

  • readPresentCurrent()
  • readPresentVoltage()
  • readCapacityUsed()
  • readElapsedTime()

RTC class

Real-time Clock.

  • getSeconds()
  • getMinutes()
  • getHours()
  • getDayWeek()
  • getDayMonth()
  • getMonth()
  • getYear()

NXTCam class

This vision subsystem tracks upto 8 colored objects. It reports coordinates of each object to Arduino. Can also be connected to a PC on USB port.

  • issueCommand()
  • sortSize()
  • selectObjectMode()
  • enableTracking()
  • disableTracking()
  • getColorMap()
  • selectLineMode()
  • resetCam()
  • sortColor()
  • sortNone()
  • getNumberObjects()
  • getBlobs()
